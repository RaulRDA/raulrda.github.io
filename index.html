Páginas web

Puedo crear páginas web desde cero con cualquier preferencia que tengas. Podrás personalizarla al máximo, no como otras páginas en las cuales sólo puedes cambiar el título.

Aplicaciones

¿Alguna vez quisiste crear tu propio juego o aplicación como las de PlayStore? No te preocupes, sólo dime los pasos que quieras, y en menos de 1 semana* tendrás la aplicación disponible tanto para Android / Iphone, como para Windows.

* La duración depende de tus preferencias (cuantas más, más se tardará).

Bots de Discord

¿Conoces la aplicación Discord? Discord es una red social en la cual puedes chatear con tus amigos (parecida a WhatsApp) pero tiene bots (como Telegram ). Puedo crearte un bot con las características que quieras, incluso con panel web para configurarlo más cómodamente.
